# PwrSimLang
PwrSimLang is used by psCARA, a Desktop ISE (Integrated Simulation Environment)
---
PwrSimLang is Copyright 2025 & Intellectual Property of Ribbon Grid an Australian Company
Ribbon Grid and psCARA are protected under Australian Trademarks 
---

The PwrSimLang is used to describe power system modeling Jobs, which are contained in *.pwrsim files. 
The language is a proprietary plain text language with a proprietary natural language interpreter and linter. 

The user can enter plain text into the psCARA software like:
'run this script pythonfile.py'
